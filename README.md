# Demo

这是一个基于 `Next.js 16`、`React 19`、`TypeScript` 和 `Tailwind CSS 4` 的初始项目，可用于快速开始 App Router 应用开发。

## 技术栈

- `Next.js 16.2.4`
- `React 19.2.4`
- `TypeScript 5`
- `Tailwind CSS 4`

## 环境要求

- `Node.js >= 20.9.0`
- 推荐使用 `nvm` 管理 Node 版本

如果你本机还没有切到 `Node 20`，可以先执行：

```bash
nvm install 20
nvm use 20
```

## 初始化项目

首次拉取项目后，先安装依赖：

```bash
nvm use 20
npm install
```

## 运行项目

启动开发环境：

```bash
npm run dev
```

启动后默认访问地址：

```text
http://localhost:3000
```

## 构建与启动

生成生产构建：

```bash
npm run build
```

启动生产环境：

```bash
npm run start
```

## 常用命令

```bash
npm run dev
npm run build
npm run start
```
